About

Code Monster from Crunchzilla is an interactive tutorial for kids that focuses on action. Code changes immediately yield visible results.

Projects start with simple boxes and colors, rapidly progressing into exciting experiments with simple animation and fractals. Important programming concepts like variables, loops, conditionals, expressions, and functions are introduced by example.

Code Monster is a gentle and fun introduction to programming concepts. It is a first step in learning to program. It is not intended to teach all of computer science and programming.

Code Monster is based in Seattle, WA. It is part of the Crunchzilla suite of game and educational projects developed by Geeky Ventures.

Gyakran Ismétlődő Kérdések

Mi a c a darabokban, mint a c.save()?

Ez egy kicsit technikai de, ha tudni akarod a részleteket, a c egy változó Code Monster hozta létre neked, aemly tartalmazza a Canvas(Vászon) 2D összefüggéseit. A kód amely létrehozza a c változót, amely rejtett, valahogy így néz ki

var c = document.getElementById('pane').getContext('2d');

Hogyan tudom menteni a saját kódomat és felhasználni a Code Monsteren kívül?

Rövid válasz: Ez nem nagyon könnyű, de másolhatod és mozgathatod a kódodat és azután betöltheted egy HTML fájlba (akár betötlheted egy különálló Javascript fájlba akár inline Javascript-ként a HTML fájlodba ).

Hosszabb válasz: Egy minimális HTML, amely működik így nézhet ki, mint ez:

<html><body>
<canvas id=pane width=400 height=300></canvas>
<script>
var c = document.getElementById('pane').getContext('2d');

// ILLESZD BE A SAJÁT KÓDODAT IDE

</script></body></html>

Ha ezt bemásolod egy fájlba, majd másolod és beilleszted a kódodat ebbe, majd a fájlt elmented egy .html-re végződő fájlnéven, be tudod tölteni a fájlt bármelyik böngészőbe.

Van még bármilyen kód, ami rejtett előttem?

Van még egy pici rejtett kód azért, hogy megpróbáljuk megállítani az animációt, amikor leckét váltasz és a kód, amit írsz úgy hangszereli, hogy megpróbálja megvédni a végtelen hurokoktól. Ezek nem szükségesek neked, hogy a kódot a Code Monsteren kívül futtasd.

Miért nem mondasz többet a Javascript szintaktikáról és a fogalom meghatározásokról?

Már számos oktatóprogram és könyv áll rendelkezésre arra, hogy megtanítsák a Javascript programozást elmagyarázva, hogy mi a változó, mi az érvényessége, terjedelme, hogy néz ki a szintaktikája az if kifejezésnek, hogyan működnek egyes műveleti jelek, és így tovább. Ha ezt akarod, számos lehetőség elérhető, számos ezek közül nagyon jó kiegészítóje a Code Monsternek.

Amit nehéz megtalálni az egy oktatóprogram, ami mókás, magával ragadó, és lehetőséget kapsz, hogy azonnal király dolgokjat építhess. Code Monster átugrik számos szintaktikát és magyarázatot, amiket te megkaphatsz máshol (használd a Google-t arre, amikor szükséges) és helyette egyenesen beleugrik a szóratkoztató részbe, amit te csinálhatsz akkor, ha programozni tanulsz.

Miért nem működik ez tökéletesen egy iPaden (vagy másik mobil eszközön)?

Code Monstert betölti számos mobil eszköz, még egy kicsit játszható is függően az eszköztől, de ez nem igazán mobilra tervezett. Miért nem?

Leginkább amiatt, mert mobilon nehéz gépelni. Nagy terjedelem írása egy mobil eszközön nehéz, és ez még rosszabb, amikor az amit írni próbálsz az egy kód és neked figyelni kell a kód egyéb részeire is.Nagy terjedelmű hagyományos kódot próbálsz egy mobil eszközön írni, az nem egy jó gyakorlás.

Részben ez az oka annak, hogy úgy tűnik még senki sem hozott létre egy alkalmazást arra, hogy kódot tanulj mobilon, de vannak jó alkalmazások arra, hogy programozási koncepciókat tanítsanak (úgy mint a Hopscotch, a Move the Turtle, és a Cargo-bot).

Tudom offline használni a Code Monstert (amikor nem kapcsolódom az Internethez)?

Természetesen! Code Monster az internetet csak a legelején használja, hogy betöltse a weboldalt, és azután sohasem használja az internetet ismét.

Így , például, betöltheted a Code Monstert egy laptopra, azután beszállhatsz egy repülőgépbe vagy kisétálhatsz egy parkba, ahol nincs internet kapcsolat, nyisd ki a ott a laptopot, és használd a Code Monstert internet nélkül. Ez még elmenti a helyedet, így te visszatérhetsz ugyanahhoz a leckéhez, amikor visszatérsz! Nem szükséges egy internet kapcsolat, kivéve a legelején, amikor a weblapot betölti.
